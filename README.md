# Amazon Homepage Clone

A front-end clone of the Amazon index page built from scratch to practice structuring a complex website layout.

## 📦 Features
* **Header Navbar:** Includes the logo, delivery location tracker, search bar, language/currency selectors, and shopping cart.
* **Sub-Navigation Panel:** Secondary menu containing quick links like "Today's Deals" and "Customer Service".
* **Main Section:** A hero announcement banner and an 8-card responsive shopping grid featuring product categories.
* **Footer:** Comprehensive directory links, currency/region badges, and a copyright panel.

## 🛠️ Tech Stack
* HTML5 (Semantic elements)
* CSS3 (Flexbox & layouts)
* FontAwesome (Icons)

## 🔧 How to Run Locally
1. Clone this repository.
2. Open the `index.html` file directly in your web browser.
